Scheduled maintenance to HPC-UGent Tier2 infrastructure - COMPLETED

Start: 2025-02-24 08:00:00
End: 2025-03-07 19:00:00
Updated: 2025-03-06 08:37:59
Level: high

Maintenance to the HPC-UGent Tier2 infrastructure is scheduled for Monday 24 (08:00 CET) till Friday 28 February 2025. This will impact ALL infrastructure and services maintained by HPC-UGent.

The entire HPC-UGent Tier2 infrastructure will be offline and unavailable during this period. This includes all workernodes as well as login and debug nodes, and the Tier2 shared storage (home, data, scratch). No jobs will be able to run on the infrastructure.

Update on 28 February 2025: All Tier2 clusters are back online and have already been running jobs for several hours. Tier2 login nodes and storage are also available.

Please be aware of the following changes:

  • Cluster skitty has been decommissioned.
  • All planned network changes have been successfully completed.
  • All clusters have been migrated to the RHEL9 operating system. Very old software modules that are no longer compatible with RHEL9 are no longer available. See https://docs.hpc.ugent.be/rhel9/ for more details.
  • New login nodes have been brought in operation: RHEL9 operating system, with hostnames gligar09 and gligar10. Old gligar07 and gligar08 login nodes are no longer available.
  • IP address changes have been implemented, as a result of major network refactoring. See https://docs.hpc.ugent.be/troubleshooting/#issues-reaching-servers-from-hpc-infrastructure for more details.
  • Default locations used by R for self-installed R packages have been changed (see https://docs.hpc.ugent.be/r-packages).
  • Linux kernel and GPU drivers have been updated on login and worker nodes.

Planned maintenance

No maintenance planned currently.
