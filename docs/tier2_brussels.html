General network outage at VUB main campus

Start: 2023-07-14 11:30:00
End: Unknown
Updated: 2023-07-14 13:18:25
Level: high

On Friday 14th July at around 11 AM, the main VUB campus suffered a generalised network outage affecting the datacenter hosting the Tier-2 HPC cluster Hydra. The main issues regarding the functioning of the cluster are:

* loss of access to the login nodes of Hydra

* many running jobs in Hydra will be severely impacted due to the loss of
connection of the cluster with the storage serving the home and data
directories of the users

The network outage at VUB campus is still ongoing. We will keep this incident up to date with any new information.
